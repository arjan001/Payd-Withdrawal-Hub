KEY ENDPOINTS
=============

Registration:
  POST http://localhost:3001/api/auth/register
  Body: {"name":"John","email":"john@test.com","password":"pass123"}
  Returns: {"message":"Account created successfully","email":"..."}

Login:
  POST http://localhost:3001/api/auth/login
  Body: {"email":"john@test.com","password":"pass123"}
  Returns: {"id":1,"name":"John","email":"john@test.com"}
  Sets: JWT token in secure HTTP-only cookie

Save Credentials:
  POST http://localhost:3001/api/settings/credentials
  Headers: Cookie: (from login)
  Body: {
    "payd_account_username":"my_account",
    "payd_username":"my_api_user",
    "payd_password":"my_api_pass"
  }
  Returns: Confirmation with account details

Admin Panel (No Login Required):
  GET http://localhost:3001/test/status
  GET http://localhost:3001/test/registered-users
  GET http://localhost:3001/test/credentials
  GET http://localhost:3001/test/accounts

TESTING QUICK COMMANDS
======================

# Test Registration
curl -X POST http://localhost:3001/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Test User","email":"test@example.com","password":"pass123"}'

# Test Login
curl -X POST http://localhost:3001/api/auth/login \
  -H "Content-Type: application/json" \
  -c cookies.txt \
  -d '{"email":"test@example.com","password":"pass123"}'

# Test Credentials
curl -X POST http://localhost:3001/api/settings/credentials \
  -H "Content-Type: application/json" \
  -b cookies.txt \
  -d '{
    "payd_account_username":"testaccount",
    "payd_username":"test_api_user",
    "payd_password":"test_pass"
  }'

TROUBLESHOOTING
===============

If API server is not running:
  export NODE_TLS_REJECT_UNAUTHORIZED=0
  source /vercel/share/.env.project
  cd /vercel/share/v0-project/artifacts/api-server
  PORT=3001 node dist/index.mjs

If database has old data:
  cd /vercel/share/v0-project/scripts
  pnpm exec tsx src/clear-data.ts

If you need to rebuild:
  cd /vercel/share/v0-project
  pnpm run build
